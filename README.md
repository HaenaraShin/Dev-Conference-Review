**💻 개발자 컨퍼런스 리뷰 ✍️**

- 👨‍💻 참석했던 개발자 컨퍼런스의 리뷰를 개인적으로 정리합니다.
- 🤷‍♂️ 개인적으로 정리하기 때문에 평어체로 정리합니다.

- 🙅‍♂️ 메모는 실제 발표와 다른 부분이 일부 있습니다.
- 🗓️ 최근 순으로 정렬합니다.
